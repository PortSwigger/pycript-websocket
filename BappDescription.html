PyCript WebSocket is a Burp Suite extension that empowers users by enabling the encryption and decryption of WebSocket messages. This extension preserves the fundamental logic of the original PyCript but is solely dedicated to WebSockets. Implement custom encryption and decryption logic using various languages including Python, Go, Node.js, C, Bash, etc., addressing diverse testing requirements in a flexible manner. This is merely another variant of the original PyCript Extension, specialized for handling WebSocket Messages.

Features

  • Encrypt & Decrypt Web Socket Messages for both server-side and client-side
  • View and Modify the encrypted Messages in plaintext
  • Enjoy complete freedom in determining encryption and decryption logic

Usage

Note: PyCript WebSocket has distinct logic for conducting encryption and decryption operations. This makes it incompatible with the demo code from the original PyCript. Despite sharing the same core concept, they differ in implementation. Do not use PyCript-Template for PyCript WebSocket.
